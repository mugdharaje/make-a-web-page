HTML Concepts

Lesson 1: The Basics of HTML

WWW

The World Wide Web in simple terms is a set of several documents which are interlinked and can be accessed via the internet. HTML stands for Hyper Text Markup Language is most commonly used document type in the World Wide Web. Major pieces of World Wide Web include A user, Browser plus computer, Internet and Servers.

The user sends a request from the computer via a browser to the servers using a protocol call HTTP and the server responds to these requests by sending documents like HTML documents that the browser displays.

HTML

HTML is made up of the following 4 pieces
1) Text Content
2) Mark up
3) References to other documents
4) Links to other pages like this webpage

HTML Tags and Elements

HTML Elements also known as tags are the key components of Hyper Text Markup Language. Usually an element starts with an opening tag, closes with a closing tag and includes the contents to be dispayed in between the tags. The tags specify how the contents are seen on the web page.

Break Tag
There are some tags that do not include a closing tag.
For eg: A break tag that breaks a sentence into 2 separate lines
Line A
Line B
Anchor Tag
Some tags have attributes.
For eg: Anchor tag uses attribute which is used for making links
Span and Div Tags
These tags include ways to attach styles to the content to adjust the way text works. The major difference between these 2 tags is that Span tag is an inline element while div tag is a block element. A block element creates an invisible block around the content unlike inline element.

Lesson 2: HTML Page Structure

Boxes

Web Page is structured as multiple boxes. This makes it easier to rearrange the boxes and hence rearrange the web Page. HTML controls the structure of web page. The browser turns the HTML tags into elements that form tree like structure known as DOM - Document Object Model

Styling of Web Page

CSS is the language that controls the style or the look of a web page. It defines the size of font, color, background, border-width for each box in a web page

DIV Tag

Boxes can be defined using Div tags. Each box is assigned a class. This makes it very easy to apply style to the web page using CSS language.

Lesson 3: Creating and Styling a Web Page

Cascading Style Sheet

CSS stands for Cascading Style Sheet. It simply means that several style sheets can be defined and all can be used to create the final page. As we keep going down through the styling sheets, the most specific rule applies and this is known as cascading.

Inheritance is a key feature in CSS. Descendent elements inherit text-related properties but not box-related properties.

Creating a Web Page

A basic rule for creating a web page based on the mock-up design is to code, test and refine.
1) Look for natural boxes.
2) Look for repeated styles and semantic elements.
3) Write the HTML.
4) Apply styles in the order from biggest to smallest.
5) Fix small things eg:Resize the window and check if the code displays the webpage as expected, look at various browser outputs etc.

Class Based Styling

It's a good practice to plan the box structure on paper, which makes it easy to enclose the contents in DIV tags and assign a class name to each similar type of box. Similar contents can be assigned the same class name.

eg: Contents within various lessons can be given a class name content-description. This allows us to apply same styling to all the elements that are defined under same class.
In our eg: we can apply same styling to the text which forms content-description for different lessons. This type of styling is known as Class based styling.

This reduces repetion and is one of the key advantages of CSS language. Avoiding repetion automatically avoids errors. If a programmer chose to repeat the style for same category HTML elements, and if there is any change to be made, the same change has to be applied to all the other same category elements which is time consuming and could result into errors.
